# Deadline Planner
Студенты ежедневно сталкиваются с большим потоком задач, дедлайнов и учебных обязательств — и часто теряют контроль над ними, наша программа создана чтобы решить эту проблему, с её помощью студенты смогут эффективно распределять свои задачи.

# Технологии
Для программы был использовал Python, его стандартные библиотеки, Tkinter.

# Использование 
Чтобы установить программу - скачайте репозиторий в ZIP, распакуйте архив, откройте папку "Deadline Planner" и запустите файл Deadline Planner.exe

# Требования 
Операционная система семейства Microsoft Windows.
